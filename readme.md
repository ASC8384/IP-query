# IP-query

## 概述

**IP地址查询系统**

一个IP地址段的数据库（文本文件），用户可以在交互环境中，输入ip地址，系统给出地址对应的国家，身份，城市，ISP等信息。

## 快速开始

### 编译

``` BASH
cmake .
make
```

### 使用

``` BASH
./bin/ipquery
```
